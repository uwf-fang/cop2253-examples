COP 2253 Java Programming
=========================
This is a repository used for demonstration in the classes. Will be updated
frequently.

+ Codes are not fully tested.
+ Some source code files are not compilable; They serve as a container of code
  snippets only.
+ File organizations

  * directory name starts with ch# represent chapters (contents related to the
    chapter # in the textbook) followed by a short name indicates the topic of
    the module
  * if the chapter number does not match, contact the maintainer and refer to
    the name

Conventions
===========
+ File names: Same as class names - UpperCamelCase
+ File name extensions: .java
+ Readme and document file format: Markdown (.md)
+ Source code conventions:

  - **NO PACKAGE** statements in the source code files. **It is an overkill to
    define project in IDE such as Eclipse or IntelliJ!**
  - Each .java file is a runnable program unless specified otherwise.
  - Use the click and run feature in your editor (VS Code, JGrasp, etc.) to run
    the programs directly from the graphical interface.
